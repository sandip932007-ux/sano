📝 Todo List App Description

Todo List App is a simple web-based task management application developed using HTML, CSS, and JavaScript. It allows users to add, view, and delete tasks efficiently through an intuitive and responsive user interface. The application helps users organize their daily activities and improve productivity by maintaining a digital list of tasks.

Key Features
Add new tasks.
Delete completed or unwanted tasks.
User-friendly interface.
Responsive design for different screen sizes.
Input validation to prevent empty tasks.
Support for adding tasks using the Enter key.
